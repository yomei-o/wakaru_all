# wakaru_all

物理好きの高校生・大学生のための読み物シリーズ「わかる」全集。
各シリーズを1フォルダにまとめています（各話は自己完結の HTML。明朝＋MathJax＋動く canvas 図＋印刷/PDF対応）。

## シリーズ

| フォルダ | シリーズ | 中身 |
|---|---|---|
| [`wakaru-ba/`](wakaru-ba/index.html) | **わかる場** | なぜ力は光速で伝わるのか。場を「無次元の比」で読み解く（第1回 遅れ／第2回 なぜ c か …） |

> 姉妹編「わかる力」「わかる宇宙論」なども、同じ構成でこのリポジトリに追加していく予定です。

## 読み方
各フォルダの `index.html` が目次です。個々の HTML はブラウザで開くだけで動きます（外部依存は MathJax の CDN のみ）。印刷は各ページの「印刷 → PDF に保存」から。

## 構成メモ
各シリーズは相対リンクで姉妹編を参照します（例：`wakaru-ba/` の目次から `../wakaru-chikara/index.html`）。そのため各シリーズは<strong>このリポジトリ直下の兄弟フォルダ</strong>として置いてください。
